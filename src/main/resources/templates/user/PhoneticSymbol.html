Phonetic Symbol

音标学习

为什么要学习音标？

①利用音标本身的读音与拼写的匹配性来进行单词的记忆和背诵会比较简单。

②音标是杜绝中文式英语最佳利器。

音标分类：

音标共有48个，其中元音20个，辅音28个。

单词按音节来划分，分为单音节词，双音节词和多音节词。单音节词中，常常分为重读开音节和重读闭音节。

英语国际音标表

短元音（7个）：
/ʌ/ /ɒ/ /ə/ /ɪ/ /ʊ/ /e/ /æ/

长元音（5个）：
/ɑ:/ /ɔ:/ /ɜ:/ /i:/ /u:/

双元音（8个）：
/eɪ/ /aɪ/ /ɔɪ/ /əʊ/ /aʊ/ /ɪə/ /eə/ /ʊə/

清辅音（11个）：
/p/ /t/ /k/ /f/ /θ/ /s/ /ʃ/ /h/ /tʃ/ /ts/ /tr/

浊辅音（17个）：
/b/ /d/ /g/ /v/ /ð/ /z/ /ʒ/ /r/ /dʒ/ /dz/ /dr/ /j/ /w/ /m/ /n/ /ŋ/ /l/

独立辅音（8个）：
/h/ /r/ /m/ /n/ /ŋ/ /l/ /w/ /j/

/i:/

长元音，嘴唇微微张开。舌尖抵下齿，嘴角向两边张开，流露出微笑表情，与E发音相同。

Example : pea（豌豆），bee（蜜蜂），eat（吃），beat（敲打），deep（深的）

/i/

短元音，嘴唇微微张开，舌尖抵下齿，舌前部抬高，嘴型扁平，干脆利落。

Example : bit（少量），tip（顶部），fit（合适的），sit（使坐）

/ɔ:/

长元音，双唇收得小而圆，并向前突出，舌身往后缩。

Example : sort（种类），talk（谈话），door（门）

/ɔ/

短元音，口腔大开，嘴张大，舌向后缩，双唇稍收圆。

Example : fond（喜欢），lock（锁），dog（狗）

/a:/

长元音，口腔打开，嘴张大，舌身放平，舌尖不抵下齿，下巴放松，轻松发音。

Example : car（小汽车），art（艺术），bath（洗澡）

/ʌ/

短元音，嘴唇微微张开伸向两边，舌尖轻触下齿，唇后部稍稍抬起。

Example : color（颜色），lucky（幸运的），but（但是）

/ɜ:/

长元音，嘴型扁平，上下齿微微张开，舌身放平，舌中部稍抬起。

Example : girl（女孩），first（第一），work（工作）

/ə/

短元音，嘴唇微微张开，舌身放平，舌中部稍抬起，口腔自然放松发声。

Example : forget（忘记），dollar（美元），a（一个）

/u:/

长元音，嘴型小而圆，微微外凸，舌头尽量后缩。

Example : move（移动），spoon（汤勺），book（书）

/u/

短元音，嘴唇张开略向前突出。

Example : do（做），could（能够），should（应该）

/e/

短元音，嘴型扁平，舌尖抵下齿，齿前部稍抬起。

Example : egg（鸡蛋），get（到达），send（送）

/æ/

短元音，嘴张大，舌尖抵下齿，嘴角尽量拉向两边，成扁平形。（梅花音，嘴裂最宽一个，腮帮酸胀，发音就像挤出来一样，感觉不适应。）

Example : fat（肥的），bag（包），fact（事实）

/eɪ/

双元音，口型由半开到合，与字母A发音相同 。

Example : same（相同的），late（迟到的），cake（蛋糕）

/əʊ/

双元音，口型由半开到小，与字母O的发音相同。

Example : show（展示），know（知道），no（不）

/aɪ/

双元音，口型由开到合，与字母I的发音相同。

Example : life（生活），kind（亲切的），hi（嗨）

/aʊ/

双元音，口型由大到小。

Example : house（房子），shout（叫喊），town（城镇）

/ɔɪ/

双元音，双唇由圆到扁，口型从开到合。

Example : boy（男孩），noise（噪音），enjoy（享受）

/ɪə/

双元音，双唇始终半开。

Example : dear（亲爱的），beer（啤酒），hear（听到）

/eə/

双元音，舌端抵下齿，双唇半开。

Example : where（哪里），share（分享），bear（熊）

/ʊə/

双元音，嘴唇由收圆到半开 。

Example : sure（确信的），your（你们的）

/p/

清辅音，双唇紧闭，然后快速张开，让气流冲出口腔，发出爆破音，但声带不震动。

Example : pea（豌豆），deep（深的），cup（杯子）

/b/

浊辅音，双唇紧闭，然后快速张开，让气体冲出口腔，发出爆破音，但声带震动。

Example : cab（驾驶室；出租汽车），beat（敲打），bit（少量）

/t/

清辅音，舌尖抵上齿龈，憋住气，然后弹开舌尖，让气流从口腔中喷出，但声带不震动。

Example : eat（吃），beat（敲打），sit（坐）

/d/

浊辅音，舌尖抵上齿龈，憋住气，然后弹开舌尖，让气流从口腔中喷出，但声带要震动。

Example : deep（深的），deal（处理），bed（床）

/k/

清辅音，舌后部抵住软腭，憋住气，然后突然离开，将气流送出，像咳嗽，但声带不震动。

Example : key（钥匙），kid（小孩），cake（蛋糕）

/g/

浊辅音，舌后部抵住软腭，憋住气，然后突然离开，将气流送出，像咳嗽，但声带震动。

Example : dog（狗），give（给），get（到达）

/f/

清辅音，上齿轻轻接触下唇，然后轻轻吹气，使气流从唇齿间通过，形成摩擦，但声带不震动 。

Example :fact（事实），feet（脚），knife（刀，匕首）

/v/

浊辅音，上齿轻轻接触下唇，然后轻轻吹气，使气流从唇齿间通过，形成摩擦，但声带震动。

Example : five（五），have（有），very（非常）

/θ/

清辅音，舌尖微微伸出，上下齿轻轻咬住舌尖送气，但声带不震动。

Example : earth（地球，陆地，土壤），thought（思考），thank（谢谢）

/ð/

浊辅音，舌尖微微伸出，上下齿轻轻咬住舌尖送气，但声带震动。

Example : than（比较），the（这个），those（那些）

/s/

清辅音，双唇微微张开，舌端靠近齿龈，舌头自然放松，气流从齿间送出，但声带不震动。

Example : best（最好的），zest（热情），yes（好）

/z/

浊辅音，上齿轻轻接触下唇，然后轻轻吹气，使气流从唇齿间通过，形成摩擦，但声带震动。

Example : zest（热情），zero（零），zoo（动物园）

/ʃ/

清辅音，双唇收圆，并稍微突出，舌头稍微上卷，舌头接近上齿龈送气，声带不震动。

Example : flash（闪现），wash（洗），shine（发光）

/ʒ/

浊辅音，双唇收圆，并稍微突出，舌头稍微上卷，舌头接近上齿龈送气，声带震动。

Example : pleasure（愉快），leisure（安逸），beige（米黄色）

/tʃ/

清辅音，双唇略微张开突出，舌尖抵住上齿龈，用力吐气，声带不震动。

Example : peach（桃子），change（改变），cheer（喝彩）

/dʒ/

浊辅音，双唇略微张开突出，舌尖抵住上齿龈，用力吐气，声带震动。

Example : change（改变），juice（果汁），joke（玩笑）

/tr/

清辅音，双唇收圆向前突出，舌尖上翘抵住上齿龈，采取发/r/的姿势，声带不震动。

Example : train（火车），tree（树），truck（火车）

/dr/

浊辅音，双唇收圆向前突出，舌尖上翘抵住上齿龈，采取发/r/的姿势，声带震动。

Example :dry（干燥），dress（穿衣；连衣裙），drag（拉）

/ts/

清辅音，舌尖先抵住上齿，堵住气流，使气流从舌尖和齿龈间送出，声带不震动。

Example : cats（猫（复数）），roots（根），fruits（水果）

/dz/

浊辅音，舌尖先抵住上齿，堵住气流，使气流从舌尖和齿龈间送出，声带震动。

Example : reads（阅读），goods（商品），beds（床（复数））

/h/

清辅音，嘴唇自然张开，自然呵气，声带不震动。

Example :have（有），hi（嗨），happen（发生）

/r/

浊辅音，舌尖向上卷起，舌头不要接触任何部位，双唇稍微突出，声带震动。

Example : bread（面包），red（红色的），run（跑）

/m/

鼻音，双唇紧闭，舌头放平，气流从鼻腔送出，声带震动。

Example : money（钱），room（房间），morning（早上），mom（妈妈）

/n/

鼻音，双唇微开舌尖抵住上齿龈，气流从鼻孔里出来，声带震动。

Example : need（需要），thin（瘦的），morning（早上），run（跑）

/ŋ/

鼻音，双唇张开，舌尖抵住下齿龈，舌后部自然往上顶住，气流从鼻腔送出，声带震动。

Example : young（年轻的），sing（唱），tongue（舌头）

/w/

半元音，双唇缩小并向前突出，舌后部抬起，嘴慢慢向两旁滑开，声带震动。

Example : wait（等待），watch（手表），water（水），wash（洗）

/j/

半元音，嘴形成微笑状，舌尖抵住下齿，舌面贴住上颚，声带震动。

Example : year（年份），younger（年轻的），yellow（黄色的），yes（好）

/l/

边音，舌尖抵住上齿龈，舌尖轻微用力弯曲，气流从舌的旁边送出，声带震动。

Example : love（爱），little（小的），letter（字母），tell（告诉）
